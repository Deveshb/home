Feature One

Insert data into your CloudBost app :

Inserting data into your CloudBoost app is very simple. Here we create a new CloudObject of type Student, set a name to John Smith and save it to the database.



Query data in CloudBoost :

Querying data from your CloudBoost app is crazy simple. Here we create a new CloudQuery object of type Student, set a name to John Smith and query it from the database.

Feature Two

Get notifications when object is inserted / updated or deleted :

Get instant alerts when data in your database is created / updated or deleted and keep all your apps in sync and in realtime. WOW your users with great user expeirence by implementing realtime.



Updating your app with realtime notifications :

CloudNotifications provide an easy way to crate custom channels, and publish data to those channels and update your app in realtime.

Feature Four

Index objects for search :

Indexing your data is easy. Every CloudObject has a property called isSearchable, You just have to set it to true and We'll automatically index your object for search.



Search your data with CloudSearch :

CloudSearch is more like a search-engine which returns your data in the order of relevance and searching for your data is as simple as querying.
